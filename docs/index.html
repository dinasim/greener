ABOUT THE APP

Helping plants and businesses grow

Greener is a smart plant care and marketplace app designed to help both regular users and businesses grow, trade, and take care of plants in smarter ways.

For plant lovers: AI-powered guidance, weather-aware tips, and smart care tracking‚Äîplus photo-based plant and disease recognition. Get help nurturing your plants and consult with experts or nearby businesses when you need advice.

For businesses: Manage inventory, publish marketplace listings, and reach nearby customers with location services‚Äîplus direct messaging to answer questions, offer guidance, and close sales.

By combining personal plant care with a vibrant community forum, Greener creates a space where plant lovers, sellers, and nurseries can exchange knowledge, connect, and thrive together.

Services

Services

01
Azure Cosmos DB

A globally distributed, multi-model database service. Cosmos DB stores all Greener user, plant, and marketplace data, enabling fast, scalable, and reliable access for both users and businesses.

02
Azure Functions

A serverless compute service that runs backend logic, scheduled notifications, and automation tasks‚Äîsuch as sending weather alerts or care reminders‚Äîwithout managing servers.

03
Azure Blob Storage

Object storage for unstructured data, used in Greener to save user-uploaded images (like plant photos), business assets, and other large files securely and efficiently.

04
Speech-To-Text

Real-time speech recognition so users can add care notes or search for plants by voice, making the app more accessible and hands-free.

05
Firebase Cloud Messaging (FCM)

Sends instant push notifications to users' web browsers and mobile devices, keeping them up-to-date with weather alerts, plant care reminders, and important app updates.

06
OpenWeatherMap API

Supplies real-time weather forecasts used in Greener's smart notifications, enabling personalized care advice and alerts for each user's plant location.

07
Marketplace API

Powers the app's buying and selling experience by managing products, seller profiles, reviews, messaging, and location services through a unified backend.

08
Google Gemini API

Provides advanced generative AI capabilities for the app, enabling natural language understanding, smart plant care suggestions, and interactive user experiences.

09
Azure Signal IR

Real-time, bi-directional messaging between app and backend‚Äîpowering live marketplace chat, instant order updates, and in-app notifications without page refreshes

10
Plant Identification API

Enables users to take or upload a photo of a plant and instantly recognize its species. The API analyzes the image via the PlantNet recognition service and returns details such as the scientific name, common name, family, and reference images ‚Äî powering the app's "Scan" feature for effortless plant discovery.
